# Finance Tracker

A full-stack Finance Tracker web app built with the MERN stack to help users record income and expenses, view summaries, and analyze monthly spending patterns.

## Live Demo
Frontend: https://finance-tracker-gamma-blond.vercel.app/

## Features
- Add, edit, and delete transactions
- Separate income and expense tracking
- Summary cards for total income, total expense, and balance
- Category-based expense analysis
- Monthly trend analytics
- Responsive UI
- Deployed frontend and backend
- Connected to MongoDB Atlas

## Tech Stack
- Frontend: React, Vite, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas, Mongoose
- Deployment: Vercel, Render

## Project Structure
- `frontend/` - React client
- `backend/` - Express API
- `render.yaml` - Render deployment config
- `vercel.json` - Vercel deployment config

## Environment Variables
### Backend
- `MONGODB_URI`
- `CLIENT_ORIGIN`
- `PORT`

### Frontend
- `VITE_API_URL`
